✦ guia de configuração

Conecte sua planilha ao ztrack em ~5 minutos. É gratuito!

passo 01

Abra o Google Sheets → Extensões → Apps Script

Apague o código existente e cole este script:

function doGet() {
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName("Banco de Dados");
  var data = sheet.getDataRange().getValues();
  var rows = [];
  for (var i = 1; i < data.length; i++) {
    var row = data[i];
    if (!row[0] || !row[1]) continue;
    var dv = row[0];
    if (dv instanceof Date)
      dv = Utilities.formatDate(dv,"America/Sao_Paulo","yyyy-MM-dd");
    rows.push({"Data":dv,"Tipo":row[1]||"",
      "Forma de pagamento":row[2]||"",
      "Categoria":row[3]||"","Descricao":row[4]||"","Valor":row[5]||0});
  }
  var saldo  = sheet.getRange("J2").getValue();
  var cartao = sheet.getRange("K5").getValue();
  return ContentService
    .createTextOutput(JSON.stringify({data:rows,saldo:saldo,cartao:cartao}))
    .setMimeType(ContentService.MimeType.JSON);
}
passo 02

Publicar como Web App

Implantar → Nova implantação
Tipo: App da Web · Executar como: Eu
Acesso: Qualquer pessoa → clique em Implantar e copie a URL.

passo 03

Cole a URL no ztrack e clique em salvar

Pronto! A planilha continua privada — só o script acessa os dados.
